# VDS Tecnologia - Site Institucional

Site institucional moderno e tecnológico da VDS Tecnologia, desenvolvido com Next.js 15, React 19 e TypeScript.

## Características

- Design moderno com glassmorphism e animações suaves
- Tema claro/escuro com toggle
- Totalmente responsivo
- Animações de scroll e micro-interações
- Sistema de partículas animadas
- Seções destacando experiência em clouds, Snowflake, Cortex AI e parceria com Cursor

## Tecnologias

- Next.js 15 (App Router)
- React 19
- TypeScript
- CSS Modules
- Lucide React (ícones)

## Como executar

```bash
# Instalar dependências
npm install

# Executar em desenvolvimento
npm run dev

# Build para produção
npm run build

# Executar produção
npm start
```

## Estrutura do Projeto

```
vdstecnologia/
├── app/              # Páginas e layouts
├── components/       # Componentes React
├── contexts/         # Contextos React (Theme)
├── lib/              # Utilitários e constantes
└── public/           # Arquivos estáticos e logos
```

## Seções

1. **Hero** - Seção principal com título impactante
2. **Sobre** - Informações sobre a empresa e parceiros
3. **Serviços** - Serviços oferecidos
4. **Tecnologia** - Stack tecnológica
5. **Parceiros** - Logos de clouds e parceiros
6. **Processo** - Metodologia de desenvolvimento
7. **Benefícios** - Vantagens competitivas
8. **Contato** - Formulário de contato
